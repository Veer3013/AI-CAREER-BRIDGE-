🚀 AI-First Career BridgeYour personal AI architect for professional identity and career planning.The AI-First Career Bridge is a comprehensive, all-in-one monolithic application (app.py) that translates raw technical skill into elite professional representation. By merging Generative AI with real-time data extraction, this system eliminates the "blank page" problem, synthesizing messy data into high-fidelity ATS resumes, coded portfolios, and strategic career roadmaps in seconds.⚡ Core ArchitectureThis project utilizes a Single-File Monolith Architecture, integrating the frontend Streamlit presentation layer seamlessly with the intelligence backend.The Engine: Operates via a high-speed Python subprocess CLI Bridge to the Gemini 3 Flash Preview model for maximum inference speed and zero-latency terminal integration.The Fusion Protocol: Simultaneously syncs live GitHub repositories (via GitHub REST API) and bypasses LinkedIn security (via Scrapetable API) to extract real-world technical data dynamically.The Render Layer: Uses coordinate-based mathematical mapping (fpdf) to generate 100% ATS-readable documents.🚀 Key Features1. 📝 AI ATS Resume BuilderConstructs high-fidelity, mathematically precise blueprints using real-time visual synchronization. It doesn't just convert HTML to PDF; it uses the FPDF library to map text to exact (X, Y) coordinates using standard fonts (Helvetica/Times) to guarantee ATS parsing. Includes 10 professional aesthetic templates (e.g., Harvard Classic, Minimalist Startup, CyberSecurity).2. 🌐 Pro Website BuilderActs as a zero-shot code generator. By analyzing your technical footprint (uploaded PDF, GitHub, and LinkedIn), it synthesizes a complete, single-file HTML5/CSS3 portfolio codebase tailored to selected design themes. Includes a "Magic Editor" to apply natural language CSS/HTML tweaks on the fly.3. 🧠 Smart Career MapperA career analyzer that mines current industry roles within any technical domain and generates exhaustive, step-by-step markdown learning trajectories and roadmaps.4. 💡 Project IdeatorGenerates high-impact project ideas based on your specific tech stack and provides detailed, actionable build guides to solidify your technical presence in the market.🛠️ Technical StackFrontend Framework: StreamlitAI Intelligence Core: Google Gemini 3 Flash (via CLI subprocess bridge)Document Synthesis: fpdf, PyPDF2Data Extraction & Linking: requests, beautifulsoup4💻 Installation & SetupPrerequisitesPython 3.8+ installed on your system.The Gemini CLI installed and authenticated on your local machine (the app relies on terminal access to the model).1.
